Important FAQ ?

What is the Document Object Model(DOM)?
The DOM is a tree-like structure that represents everything on a web page all the elements, text, attributes, etc.
How do you select an element from the DOM?
The DOM lets JavaScript select, read, and modify webpage elements using methods like getElementById, querySelector, and more.
What is the event delegation in the context of the DOM, and why is it useful?
Event delegation is a technique where instead of attaching event listeners to multiple child elements, you attach a single listener to a parent element. The parent listens for events bubbling up from its children, and handles them based on conditions.
How do you manipulate an element's attributes and styles using the DOM?
Use DOM methods to get, set, or remove attributes, and modify styles directly or by adding/removing CSS classes.
